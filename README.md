AnsibleRole-Gitlab
=========

This role is designed to install Gitlab in a self-hosted context, and optionally install / turn on LDAP for Gitlab

Requirements
------------

None

Role Variables
--------------

- **var1**: EXPLANATION
  - Defaults to *default1*
- **var2**: EXPLANATION
  - Defaults to *default2*

Dependencies
------------

No role dependencies.

Example Playbooks
----------------

```yaml
# EXPLANATION
    - role: 'irontooch.gitlab'
      vars:
        var1: "myapp"
```

```yaml
# EXPLANATION
    - role: 'irontooch.gitlab'
      vars:
        var1: "myapp"
        var2: "myapp"
        var3: "myapp"
        var4: "myapp"
```

```yaml
# EXPLANATION
    - role: 'irontooch.gitlab'
      vars:
        var1: "myapp"
        var2: "myapp"
        var3: "myapp"
```
